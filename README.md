# Elvira Strategy

Sitio web de Elvira Strategy - Estrategia, Equidad e Impacto.

## Estructura

- `index.html` - Página principal
- `consultoria.html` - Página de consultoría
- `cursos.html` - Página de cursos
- `styles.css` - Estilos (actualmente en línea en cada HTML)

## Tecnologías

- HTML5
- CSS3 (Inter font)
- JavaScript vanilla

## Desarrollo

Abre cualquier archivo HTML en un navegador o usa Live Server para desarrollo local.

## Despliegue

Este proyecto está configurado para desplegarse en Vercel.
